# SpecView

A simple web-based spectrogram visualizer.

**Live Demo**: https://richermans.github.io/SpecView/

## Features

- **Spectrogram Visualization**: View audio files as spectrograms.
- **Audio Playback**: Play, pause, stop, and seek.
- **Auto-Grouping**: Files with matching names and different suffixes (e.g., `_orig`, `_pred`, `_gt`) are automatically grouped side-by-side for easy comparison.

## Usage

1.  Open `index.html` in a browser.
2.  Drag & drop audio files or click to browse.
3.  Click the spectrogram to seek.
4.  Use the toolbar for playback controls.

## Keyboard Shortcuts

- `Space`: Play/Pause
- `Shift + Space`: Play from start
- `Escape`: Stop
- `←` / `→`: Seek backward/forward 2 seconds
